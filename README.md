loop-qwerty
===

Qwerty keyboard bindings for [loop-grid](https://github.com/mmckegg/loop-grid).

## Grid

```
Q W E R T Y U I O P
A S D F G H J K L ;
Z X C V B N M , . /
```

## Control Keys

- **store**: space
- **hold**: enter
- **repeat**: 1-8 (none, 1, 2/3, 1/2, 1/3, 1/4, 1/6, 1/8)
- **clear/flatten**: backspace
- **undo**: -
- **redo**: =
- **halve length**: [
- **double length**: ]
